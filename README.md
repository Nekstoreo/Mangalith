## 📦 Instalación



```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/mangalith.git
cd mangalith

# Iniciar con Docker Compose
docker-compose up -d

# Iniciar el frontend
cd frontend
pnpm dev

# Iniciar el backend
cd backend
pnpm dev

# 🚀 Acceder a la aplicación en http://localhost:3000
```
